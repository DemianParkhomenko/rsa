# Build and run
- From Visual Studio Code:
  - Build
    - Task: Run Build Task -> gcc build project
  - Run
    - Debug: Start without debugging
    - or just use UI (run project)
- CLI 
  - Build
    - gcc -g ./src/*.c ./tests/*.c -o main.out
  - Run
    - ./main.out
